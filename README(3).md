# Interactive Data Visualization and Storytelling using Power BI

## Project Overview
This project demonstrates how business data can be transformed into meaningful insights using Microsoft Power BI. The dashboard provides an interactive and visually appealing analysis of sales, profit, customer segments, product categories, and regional performance.

## Problem Statement
Businesses generate large amounts of sales data every day, making it difficult to identify trends and performance using spreadsheets alone. This project addresses that challenge by creating an interactive dashboard that simplifies data analysis and highlights important business insights.

## Objectives
- Build an interactive Power BI dashboard.
- Analyze sales and profit performance.
- Compare regional and category-wise performance.
- Identify business trends and top-performing products.
- Present data in a clear and meaningful visual format.

## Dataset
- Superstore Sales Dataset (CSV)

## Tools & Technologies
- Microsoft Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV

## Dashboard Components
- KPI Cards (Total Sales, Total Profit, Total Orders)
- Sales by Category
- Profit by Category
- Sales by Region
- Monthly Sales Trend
- Customer Segment Analysis
- Interactive Slicers

## Key Insights
- Highest-performing regions by sales
- Most profitable product categories
- Monthly sales trends
- Customer segment performance
- Top-selling products

## Skills Demonstrated
- Data Cleaning
- Data Modeling
- Dashboard Design
- Business Intelligence
- Data Storytelling
- DAX
- Data Visualization

## Conclusion
This Power BI dashboard converts raw sales data into actionable business insights for better decision-making.
